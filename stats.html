# المادة والصف الزيارات الشعبية
تعذَّر قراءة الأرقام — تعرض الشارات التالية العدد مباشرة:
💡 هل تريد إحصاءات أكثر تفصيلاً؟ (مجاناً)
أضف Google Analytics 4 بخطوتين:
  1. سجّل في analytics.google.com ← أنشئ Property → Web → انسخ Measurement ID (يبدأ بـ G-)
  2. افتح config.js ← ضع الـ ID في GA_ID = "G-XXXXXXXX" ← أعِد الرفع
ستحصل على: خريطة دول الزوّار · نوع الجهاز · وقت الذروة · مسارات الطلاب.

أو جرّب GoatCounter (خصوصيّة كاملة، GDPR، مجاني) — ضع كوده في GOAT_CODE بـ config.js.
